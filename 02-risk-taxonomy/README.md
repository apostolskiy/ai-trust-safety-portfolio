# Risk Taxonomy

This project will define a structured taxonomy for classifying risks in AI-generated content.

## Planned Categories

- Harmful instructions
- Misinformation
- Bias and stereotyping
- Manipulation
- Impersonation
- Synthetic media deception
- Unsafe advice
- Inappropriate content
- Hallucinations
- Platform abuse

## Planned Fields

- Risk category
- Description
- Example
- Severity level
- Potential user harm
- Recommended mitigation
